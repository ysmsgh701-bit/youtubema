CapCut AI — 스크립트→영상
무료 | 설치됨 | 약 10~15분
1
대본 Word 파일 준비

대본 생성 탭에서 주제 입력 → Word 저장 클릭
또는 아래에서 바로 복사하세요

2
CapCut 실행 → AI 도구

CapCut 열기 → 상단 메뉴 "AI 도구" 또는
새 프로젝트 → "스크립트로 동영상 만들기"

CapCut → AI 도구 → 스크립트로 동영상 만들기
3
대본 붙여넣기

대본 텍스트를 입력창에 붙여넣기
섹션별(훅/본론/마무리)로 나눠서 넣으면 장면 구성이 더 잘 됨

4
스타일 선택

트랙별 권장 스타일:
재무AI 비즈니스/뉴스 스타일 → 차트/그래프 클립
TFT 게이밍/다이나믹 스타일 → 활기찬 전환효과

5
AI 보이스 선택 (텍스트→음성)

CapCut 내장 TTS → 한국어 → "뉴스 앵커" 또는 "여성 전문가"
볼륨: 배경음악 20% / 음성 80% 비율 권장

6
자막 자동 생성 + 내보내기

텍스트 탭 → 자동 캡션 생성
내보내기: 1080p / MP4 → inbox/finance/ 저장
→ 파이프라인이 자동으로 YouTube 업로드 처리

Canva — 프레젠테이션→영상
무료 | 설치됨 | 약 15~20분
1
Canva 실행 → 새 프레젠테이션

Canva 열기 → 프레젠테이션 (16:9) 선택
검색창에 트랙 키워드 입력해서 템플릿 선택

재무AI: "비즈니스 발표" 템플릿
TFT: "게이밍" 또는 "YouTube 썸네일" 템플릿
2
Magic Write로 슬라이드 생성

좌측 앱 → Magic Write 클릭
대본 텍스트 붙여넣기 → 슬라이드 자동 분배

3
AI 이미지 삽입

각 슬라이드에 앱 → Text to Image로 장면 이미지 생성
하루 50회 무료 (무료 플랜 기준)

4
프레젠테이션 녹화 (Canva 내장)

공유 → 프레젠테이션 녹화
화면 + 웹캠 + 마이크 동시 녹화 가능
직접 대본을 보며 읽기 — 가장 자연스러운 영상

공유 → 프레젠테이션 녹화 → 화면+카메라 선택 → 녹화 시작
5
MP4 내보내기

공유 → 다운로드 → MP4 동영상
→ inbox/{트랙}/ 폴더에 저장
→ 파이프라인 자동 처리

OBS Studio 기본 설정
1
설치 및 초기 설정

OBS Studio 공식 사이트에서 설치 후 자동 구성 마법사 실행
용도: 녹화 최적화 선택 → 해상도 1920×1080, 60fps

2
인코더 설정 (설정 → 출력 → 녹화)

인코더: H.264 (x264) 또는 NVIDIA NVENC
품질: CRF 18 (고화질) / CRF 23 (용량 절충)

설정 → 출력 → 녹화 탭 → 품질 프리셋: "높음"
3
마이크 설정 (음성 믹서)

마이크 레벨 -12 ~ -6 dB 유지 (초록~노랑 구간)
노이즈 억제 필터 추가 권장 → 소스 우클릭 → 필터 → + → RNNoise

4
녹화 파일 저장 경로 설정 (핵심!)

설정 → 출력 → 녹화 경로를 아래 폴더로 지정하면
녹화 완료 즉시 파이프라인이 자동 실행됩니다

C:\...\youtube\inbox\finance\
5
핫키 설정 (설정 → 단축키)

녹화 시작: F9    녹화 중지: F10
녹화 중 확인: 우하단 빨간 ● 표시
